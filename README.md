# مجموعة أبو باشا — موقع ثابت

موقع تعريفي لمجموعة أبو باشا للثروة الداجنة ومستلزمات المزارع.  
HTML + CSS + JavaScript بدون إطار عمل — جاهز للرفع على أي استضافة ثابتة.

## الملفات

| الملف | الوظيفة |
|-------|---------|
| `index.html` | الصفحة الرئيسية |
| `admin.html` | لوحة التحكم (أسعار، فئات، صور، عملاء) |
| `data.js` | أرقام التواصل والبيانات الافتراضية |
| `storage.js` | التخزين المحلي + دوال الأسعار |
| `app.js` | منطق الموقع الرئيسي |
| `admin.js` | منطق لوحة التحكم |
| `style.css` / `antigravity.css` | التنسيقات |
| `admin.css` | تنسيق لوحة التحكم |
| `assets/tarboosh.png` | أيقونة الطربوش (فافيكون + الهيرو) |

## التشغيل محلياً

1. افتح المجلد على جهازك.
2. انقر مرتين على `index.html` أو استخدم Live Server في VS Code/Cursor.
3. للإدارة: افتح `admin.html` في نفس المتصفح.

> **مهم:** البيانات تُحفظ في `localStorage` للمتصفح. بعد الرفع على الاستضافة، افتح `admin.html` على الدومين الجديد واحفظ التعديلات هناك.

## تعديل أرقام التواصل

عدّل في `data.js`:

```javascript
const WHATSAPP_NUMBER = '201070780092';
const PHONE_NUMBERS   = ['201070780092', '201001887670', '201211635266'];
const FACEBOOK_URL    = 'https://www.facebook.com/...';
```
